Directives du Rituel

Étape 1 : Choix de Liaison

Sélectionnez le mode de couplage souhaité. Le Code QR nécessite un second écran, tandis que le code de jumelage utilise une simple notification.

Étape 2 : Synchronisation

Ouvrez WhatsApp sur votre téléphone principal > Appareils connectés > Connecter un appareil. Entrez le code ou scannez le QR.

Étape 3 : Session Initialisée

Une fois couplé, votre identifiant unique (Session ID) sera envoyé directement dans vos messages privés WhatsApp de manière ultra-sécurisée.
